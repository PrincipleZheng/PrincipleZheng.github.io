hexo generate 生成静态文件，hexo server 本地localhost测试，hexo deploy 部署远程。

另一个方法: （个人比较推荐）

（1）更改根目录的配置文件，在全局配置文件（hexo/_config.yml)中将post_asset_folder设置为true

（2）hexo new “XXX”时，_post里面会自动建立一个新的文件夹“XXX”，这样，我们只需要将图片放到新的文件夹里面就好了。

这样文件组织更加整洁一些。

注意：这种方式插入图片不能使用![]()格式，而是：<img src="XXX/图片名称" style="" />
